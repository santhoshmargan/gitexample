Welcome to Git commands

git init -- to make current folder as local repository

git add file_name -- to add file to the staging area

git commit -m "commit message" -- to commit the changes to the local repo

git push origin master -- to push commited files in local to remote repo

git pull origin master -- to get remote files in local repo

git branch branch_name -- to create new branch

git checkout master -- to switch to master

git checkout branch_name -- to switch to particular branch

git -d branch_name -- to delete the particular branch
